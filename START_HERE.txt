🚀 PRÓXIMOS PASOS (DEBES HACER ESTO AHORA)

OPCIÓN 1: WINDOWS (RECOMENDADO)
────────────────────────────────
1. Abre PowerShell o CMD en la carpeta del proyecto
2. Ejecuta:
   .\setup.bat

3. Espera a que termine (instalará dependencias)
4. Una vez completado:
   npm run dev

5. ¡Abre http://localhost:5173 en tu navegador!


OPCIÓN 2: macOS/LINUX
─────────────────────
1. Abre Terminal en la carpeta del proyecto
2. Ejecuta:
   chmod +x setup.sh && ./setup.sh

3. Espera a que termine
4. Una vez completado:
   npm run dev

5. ¡Abre http://localhost:5173 en tu navegador!


OPCIÓN 3: MANUAL (Si los scripts no funcionan)
───────────────────────────────────────────────
1. Ejecuta en terminal:
   npm install

2. Luego:
   npm run dev

      variants={fadeInUp}

    >
      <h2 className="text-2xl font-bold text-white">

      </h2>
    </motion.div>

🔧 TROUBLESHOOTING

"npm: command not found"
  → Instala Node.js desde https://nodejs.org/

"Port 5173 already in use"
  → npm run dev -- --port 5174

"Module not found"
  → Asegúrate de que npm install se completó correctamente
